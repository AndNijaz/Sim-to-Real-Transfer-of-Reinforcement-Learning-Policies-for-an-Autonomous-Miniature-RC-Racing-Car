# Real-World Deployment Code

This folder contains the real-world deployment code for the autonomous miniature RC racing car.

Main areas:

- `Arduino`: Arduino actuator controller code and PWM protocol notes.
- `Configs`: configuration, requirements, environment, and working setup notes.
- `Raspberry_Pi`: Raspberry Pi deployment scripts.

The Raspberry Pi folder includes autonomous inference scripts, camera and perception utilities, core vehicle control code, manual driving scripts, and calibration/test helpers. The Arduino sketch `arduino_servo_esc_controller.ino` supports actuator control for the steering servo and ESC.

Manual, calibration, and test scripts are supporting evidence for the real-world setup. Autonomous inference scripts and core control scripts are the main deployment evidence.
