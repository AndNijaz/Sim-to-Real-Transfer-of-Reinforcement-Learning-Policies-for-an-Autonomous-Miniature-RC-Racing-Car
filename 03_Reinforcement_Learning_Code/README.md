# Reinforcement Learning Code

This folder contains the reinforcement learning implementation code used for the thesis. PPO and SAC are implemented separately.

Main folders:

- `PPO`: PPO training, run/evaluation, and environment code.
- `SAC`: SAC training and run/evaluation code.
- `Utilities`: utility folder retained for supporting code or helper material.

The PPO folder includes `train_ppo.py`, `run_ppo.py`, and `env/buggy_env.py`. The SAC folder includes `train_sac.py`, `run_sac.py`, and `train_sac2.py`. Existing `.txt` README files are also present for PPO and SAC notes.
