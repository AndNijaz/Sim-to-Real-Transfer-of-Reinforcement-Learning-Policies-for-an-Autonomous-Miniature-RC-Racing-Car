# Results, Logs, and Graphs

This folder contains the main numerical and visual evidence used to support the thesis.

Main areas:

- `PPO`: PPO training summaries, TensorBoard exports, and deterministic evaluation logs.
- `SAC`: SAC 10-minute deterministic evaluation logs.
- `Webots_10min_Evaluation`: combined Webots evaluation summary containing both PPO and SAC comparison data.
- `Thesis_Figures_Final`: final figures used in the thesis.

PPO training logs support the PPO training curves. SAC logs support the 10-minute deterministic Webots evaluation. The combined Webots evaluation summary contains both PPO and SAC comparison data.
